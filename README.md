# Mythical Critters
#### But in Python

### Installation & Setup

* Fork the repo
* Clone it locally
* cd into directory
* `source env/bin/activate` to get virtual environment running
* If you don't have pytest:
  *  `pip3 install pytest` or

* `python3 -m pytest tests/mythical_creatures/whatever_test.py` to run test
